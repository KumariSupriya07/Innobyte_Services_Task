# Kingsukh Guest House Website (Internship Task)

This project is a responsive website created as part of my internship task.  
It showcases hotel details, rooms, services, gallery, and a contact form.  

## Features
- Responsive header and navigation
- Room listing and booking form
- Gallery with images
- Contact form with WhatsApp/Instagram integration

## Tech Stack
- HTML
- CSS
- JavaScript

## Setup
Clone this repository and open `index.html` in your browser.
